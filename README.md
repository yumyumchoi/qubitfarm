qubitfarm
=========

Qubit Farm is an iOS app where users can connect to a world full of pre­generated unique plots. Each plot 

has a finite number of qubits.  Quibits are the game’s currency. 

Each plot contains a prefixed number of patches which grow over time.  When tapped a random number of 

Qubits are harvested.  Each plot has a predetermined growth rate, harvest yield potential, patch 

configuration and max number of harvestable qubits.

A user can purchase additional plots through a market interface. Each plot has a value based on its

potential yield of qubits. A player can own up to 9 plots at a time.  The player can swipe and play between 

plots.

When all plots in the world are depleted the game ends.  The winner is declared based on cubits harvested.
